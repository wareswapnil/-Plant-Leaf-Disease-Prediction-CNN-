## 📌 Project Overview

This project presents a **Smart Greenhouse System integrated with a Deep Learning model (CNN)** to detect plant leaf diseases from images. The system allows users to upload leaf images, perform preprocessing, and obtain real-time disease predictions along with recommended treatments.

The solution is designed to assist in **early disease detection, smart farming, and improved agricultural productivity**.

---

## 🎯 Key Features

* 📷 Image Upload for plant leaf analysis
* 🧪 Image Preprocessing (grayscale, thresholding)
* 🧠 Disease Detection using CNN model
* 💊 Medicine Recommendation for detected disease
* 💰 Estimated treatment cost display
* ⚡ Fast prediction with execution time
* 🖥️ User-friendly desktop interface

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** TensorFlow / Keras, OpenCV, NumPy, Pandas, Matplotlib
* **Model:** Convolutional Neural Network (CNN)
* **Interface:** Python GUI (Tkinter / Desktop UI)

---

## ⚙️ System Workflow

1. Select plant leaf image
2. Perform image preprocessing
3. Extract features using CNN
4. Predict disease category
5. Display:

   * Disease name
   * Recommended medicine
   * Estimated cost
   * Execution time

---

## 🧠 Model Details

* CNN-based image classification model
* Trained on labeled plant leaf dataset
